[ STEP 1 ] IDENTIFY ALL WIRELESS DEVICES
// LIST ALL INTERFACES
$ iwconfig $ ip link show CLICK TO COPY
// LIST USB WIRELESS ADAPTERS
$ lsusb $ iw dev CLICK TO COPY
// CHECK DRIVERS LOADED
$ lsmod | grep -E "88XX|8188|8192|mt79" CLICK TO COPY
// CHECK INTERFACE CAPABILITIES
$ iw phy phy0 info | grep -A 10 "Supported interface" CLICK TO COPY
INTERNAL WIFI
MediaTek MT7921  |  Driver: mt7921e  |  wlan0
Internal card does NOT support monitor mode. Use for normal internet only.
// VERIFY IT IS WORKING
$ iwconfig wlan0 $ ping -c 3 google.com CLICK TO COPY
// EXPECTED OUTPUT
wlan0 IEEE 802.11 ESSID:"YourSSID" Mode:Managed Access Point: xx:xx:xx:xx ^^^ Managed = normal, this is correct
ALFA AWUS036ACH
RTL8812AU  |  Driver: 88XXau (DKMS)  |  wlan1
// VERIFY DRIVER LOADED
$ lsmod | grep 88XX $ iwconfig wlan1 CLICK TO COPY
// ENABLE MONITOR MODE
$ sudo airmon-ng check kill $ sudo airmon-ng start wlan1 CLICK TO COPY
// VERIFY MONITOR MODE ACTIVE
$ iwconfig wlan1mon ^^^ look for Mode:Monitor CLICK TO COPY
// TEST PACKET INJECTION
$ sudo aireplay-ng --test wlan1mon CLICK TO COPY
// DISABLE MONITOR MODE
$ sudo airmon-ng stop wlan1mon $ sudo systemctl restart NetworkManager CLICK TO COPY
TP-LINK TL-WN722N V2/V3
RTL8188EUS  |  Driver: 8188eu  |  wlan1 or wlan2
// VERIFY DRIVER LOADED
$ lsmod | grep 8188eu $ iwconfig CLICK TO COPY
// ENABLE MONITOR MODE (METHOD 1)
$ sudo airmon-ng check kill $ sudo airmon-ng start wlan1 CLICK TO COPY
// ENABLE MONITOR MODE (METHOD 2 - MANUAL)
$ sudo ip link set wlan1 down $ sudo iw dev wlan1 set type monitor $ sudo ip link set wlan1 up CLICK TO COPY
// VERIFY MONITOR MODE ACTIVE
$ iwconfig wlan1mon ^^^ look for Mode:Monitor CLICK TO COPY
// TEST PACKET INJECTION
$ sudo aireplay-ng --test wlan1mon CLICK TO COPY
NETIS WF2111
RTL8192CU  |  Driver: 8192cu  |  wlan1 or wlan2
// VERIFY DRIVER LOADED
$ lsmod | grep 8192 $ iwconfig CLICK TO COPY
// ENABLE MONITOR MODE
$ sudo airmon-ng check kill $ sudo airmon-ng start wlan1 CLICK TO COPY

$ sudo ip link set wlan1 down $ sudo iw dev wlan1 set type monitor $ sudo ip link set wlan1 up CLICK TO COPY

[ TROUBLESHOOT ] COMMON ISSUES & FIXES
SYMPTOM LIKELY CAUSE FIX
Device not showing in iwconfig Driver not loaded or device not plugged in lsusb to check USB, then lsmod | grep 88XX
Monitor mode won't start NetworkManager interfering sudo airmon-ng check kill then retry
Injection test fails Wrong driver or wrong interface name Check iwconfig for correct interface name (wlan1mon vs wlan1)
Interface disappears after airmon Interface renamed by airmon-ng Run iwconfig - look for wlan1mon or similar
No internet after monitor mode NetworkManager was killed sudo airmon-ng stop wlan1mon then sudo systemctl restart NetworkManager
rtl8xxxu or r8188eu still loading Blacklist not applied yet sudo reboot - blacklist only applies after reboot
DKMS module not loading after kernel update DKMS rebuild failed sudo dkms autoinstall then sudo reboot
